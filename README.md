# 🚗 Motor Insurance Premium Prediction using Machine Learning

## 📌 Project Overview
This project predicts **motor insurance premiums** using machine learning models.  
It focuses on improving pricing accuracy by analysing structured insurance data and applying advanced regression techniques.

---

## 🎯 Objectives
- Develop ML models for premium prediction  
- Compare multiple regression algorithms  
- Improve performance using feature engineering  
- Evaluate models using standard metrics  

---

## ❓ Research Questions
- Which ML model provides the best accuracy?  
- What features influence premium prediction most?  
- Can ML outperform traditional methods?  

---

## 📂 Dataset
- 📍 Source: Kaggle  
- 📄 Type: CSV (structured data)  
- 📊 Features: insured value, vehicle type, usage, engine capacity, seats  
- 🎯 Target: Premium (continuous variable)  

---

## 🔍 Exploratory Data Analysis (EDA)
- Analysed premium distribution  
- Explored feature relationships  
- Used correlation heatmap  
- Identified key patterns and trends  

---

## ⚙️ Data Preprocessing
- Handled missing values  
- Feature engineering (vehicle age, policy duration)  
- One-Hot Encoding for categorical data  
- Built pipeline using Column Transformer  
- Train-test split (80:20)  

---

## 🤖 Models Used
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  
- HistGradientBoosting Regressor  

---

## 📊 Evaluation Metrics
- 📉 MAE (Mean Absolute Error)  
- 📈 RMSE (Root Mean Squared Error)  
- 📊 R² Score  

---

## 📈 Results
- Ensemble models performed best  
- Random Forest & Gradient Boosting achieved highest accuracy  
- Feature engineering improved performance  
- Models captured complex relationships  

---

## ⚡ Hyperparameter Tuning
- 🔧 Grid Search / Random Search with cross-validation  
- Tuned parameters:
  - max_depth  
  - min_samples_split  
  - min_samples_leaf  
  - n_estimators  
  - max_features  

✅ Improved accuracy and reduced overfitting
